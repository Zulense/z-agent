# z-agent

1. clone the repo
    `git clone https://github.com/Zulense/z-agent.git`

3. activate virtual envirnoment.
    `source .z-agent/Scripts/activate`

2. install the dependencies.
    `pip install -r requirement.txt`

3. create `.env` file in base folder and put the gemni `API_KEY` (make sure name are matched)

4. run the python file.
    `python main.py`
